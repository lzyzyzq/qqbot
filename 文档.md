##编写方法
> 插件目录:plugin/
> 在插件目录下创建php文件
> 如:plugin/test.php
> 内容如下↓(群聊/私聊例子)

```php
<?php
if (消息 == "测试") {
  文字("收到来自".用户."的消息");
}
?>
```

## 变量
```php
<?php
用户 = 用户ID
来源 = 来源(群号/用户ID)
消息来源 = 群聊/私聊/加群/退群
消息 =消息内容(群聊/私聊)
?>
```

## 消息函数
```php
文字(内容)
图片(链接,文字[选填])
语音(链接)
视频(链接)
按钮(按钮ID)
文卡(["text"=>"文字"],["text"=>"链接","url"=>"跳转链接"]) //以此类推
大图("大标题","小标题","图链");
跳转卡("标题","介绍","图链","跳转链接");
流式(内容,内容) //依次类推
撤回(ID)
文件(链接)
MD(内容)
```

## 官机函数
```php
头像(用户ID)
BOT信息()
```

## 辅助函数
```php
读(路径,键,值)
写(路径,键,值)
curl(链接,访问方式[GET/POST/PUT/DELLET],header,参数)
二维码(内容)
域名大写(消息)
markdown转html(MD内容)
邮箱(标题,内容,收信人[例:10001@qq.com],发信人[例:10001@qq.com],SMTP授权码[只支持QQ])
HTML转图(HTML,高,宽)
```

## 画布函数
```php
<?php
$h=new 画布();
$h->创建(宽,高,HEX码[选填])
$h->贴图($画布,路径/链接,X,Y,宽,高,透明度[选填])
$h->文字($画布,文字内容,文字大小,x,y,HEX颜色,字体路径,角度[选填])
$h->直线($画布,起点x,起点y,终点x,终点y,颜色)
$h->矩形($画布,左上x,左上y,右下x,右下y,颜色)
$h->填充矩形($画布,左上x,左上y,右下x,右下y,颜色)
$h->圆($画布,x,y,宽,高,颜色)
$h->填充圆($画布,x,y,宽,高,颜色)
$h->输出($画布) 输出为PNG
$h->二进制输出($画布) 输出图片原始内容(用于官机发图)
$h->销毁($画布)
?>
```
